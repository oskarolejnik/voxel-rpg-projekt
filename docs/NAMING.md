# Nazwa gry — kandydaci i clearance

> Zasada: silna, dystynktywna nazwa, **bez** „Cube World" i bliskich wariantów
> („Cubic World", „Cube Realm", „Cube World 2" itd.). Test prawny = **prawdopodobieństwo
> wprowadzenia w błąd**, nie tylko identyczny ciąg znaków. Nazwy generyczne („cube",
> „voxel") wpleć w unikalną całość. Na razie używamy nazwy roboczej; ostateczną wybierzemy
> po clearance.

> To tylko punkt startu. Najlepiej, gdy nazwę wymyślisz/poczujesz Ty — ja pomogę
> przepuścić ją przez clearance.

## Checklista clearance (zrób WSZYSTKIE dla finalnej nazwy, zapisz datę i wyniki)

- [ ] **EUIPO / TMview** — https://www.tmdn.org/tmview/ (UE, ~60 mln znaków)
- [ ] **USPTO TESS/Search** — jeśli celujesz w rynek USA
- [ ] **DPMA** — Niemcy (gdyby istotne)
- [ ] **Steam** — wyszukaj tytuł (Valve zakazuje mylących metadanych)
- [ ] **itch.io / Epic / App Store / Google Play** — kolizje tytułów
- [ ] **Google + YouTube** — znaki niezarejestrowane (common-law)
- [ ] **Domena** `.com` + uchwyty social (zarezerwuj wcześnie, gdy wolne)

## Po wyborze (opcjonalnie, pod komercję)

- [ ] Zgłoszenie własnego znaku w **EUIPO** — klasa nicejska **9** (oprogramowanie/gry), ew. **41** (rozrywka)
- [ ] Rejestracja domeny + kont social
- [ ] Zachowaj **datowany zrzut** wyników wyszukiwań (dowód dobrej wiary)
